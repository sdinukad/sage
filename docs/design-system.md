# Design System

Sage's visual identity is built on a foundation of clarity, accessibility, and modern aesthetics. Our design system ensures a consistent experience across all platforms.

## Design Philosophy

- **Chat-First**: The conversation is the heart of the app. Everything revolves around the chat interface.
- **Micro-Animations**: Subtle transitions and hover effects make the app feel alive and responsive.
- **Glassmorphism**: A modern, translucent look that provides depth and focus.
- **Accessibility**: High-contrast text and intuitive layouts for all users.

---

## Color Palette

Sage uses a system of semantic color tokens that adapt to both **Light** and **Dark** modes.

| Token | Light Mode | Dark Mode | Purpose |
| :--- | :--- | :--- | :--- |
| `background` | `#ffffff` | `#0a0a0a` | Main background color. |
| `foreground` | `#171717` | `#ededed` | Primary text color. |
| `primary` | `#2563eb` | `#3b82f6` | Brand and CTA elements. |
| `secondary` | `#f1f5f9` | `#1f2937` | Less prominent elements. |
| `accent` | `#8b5cf6` | `#a78bfa` | High-importance highlights. |
| `success` | `#22c55e` | `#4ade80` | Confirmed actions. |
| `error` | `#ef4444` | `#f87171` | Problematic states. |

---

## Typography

We use **Inter** as our primary typeface for its readability and versatility.

- **Headings**: Semi-bold to Bold, providing clear hierarchy.
- **Body**: Regular, optimized for long-form reading and status updates.
- **Monospace**: Used for technical details and code snippets.

---

## Core Components

### 1. Chat Bubble
The fundamental unit of conversation.
- **User Bubble**: Right-aligned, primary brand color.
- **AI Bubble**: Left-aligned, secondary background color with a subtle glass effect.

### 2. Transaction Card
A card-based view for individual expenses with icons for categories.

### 3. Dashboard Summary
High-level overview cards showing monthly budget vs. actual spend.

---

## Responsive Design

Sage is designed to be mobile-first and fully responsive.
- **Mobile**: Single-column chat-focused layout.
- **Desktop**: Multi-column layout with persistent sidebar for history and summaries.
